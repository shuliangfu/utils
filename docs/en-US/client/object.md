# Client Object

> Client object utilities; re-exports the server implementation.

[![JSR](https://jsr.io/badges/@dreamer/utils)](https://jsr.io/@dreamer/utils)
[![License: Apache-2.0](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://www.apache.org/licenses/LICENSE-2.0)

---

## Overview

Client object utilities match the server module. Same API, re-exported for
browser.

---

## Features

Same as server: `deepClone`, `merge`, `deepMerge`, `get`, `set`, `has`,
`deletePath`, `pick`, `omit`, `isEqual`, `isDeepEqual`.

---

## Installation

```bash
deno add jsr:@dreamer/utils
```
