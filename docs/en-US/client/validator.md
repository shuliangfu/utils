# Client Validator

> Client validator utilities; re-exports the server implementation.

[![JSR](https://jsr.io/badges/@dreamer/utils)](https://jsr.io/@dreamer/utils)
[![License: Apache-2.0](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://www.apache.org/licenses/LICENSE-2.0)

---

## Overview

Client validator utilities match the server module. Same API, re-exported for
browser. Full compatibility for form and API validation.

---

## Features

Same as server: `validate`, `validateAsync`, `validateAll`, `string`, `number`,
`boolean`, `email`, `url`, `object`, `array`, custom rules, transform, default,
when, messages. Supports `ValidateOptions` (e.g. `options.messages`) for custom
error messages; default messages are in English when options are not passed.
